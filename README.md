# Portfolio OS — Phase 6 Real Portfolio Content

Bu paket Phase 5'in üstüne gerçek portfolyo içeriğini ekler.

Kaynak alınan mevcut portfolyo:
https://4rc4.github.io/portfolio/

## Eklenenler

- Gerçek profil bilgileri
- Gerçek About Me içeriği
- Gerçek proje listesi
- Catudy, Catudy Website, Dr. Hicran Ünlü Website, Unity Story Game, Interactive Portfolio
- Eğitim ve iş geçmişi
- Skills / tools / languages / hobbies
- Contact uygulaması
- VFS içinde gerçek portfolyo klasörleri
- Project Viewer içinde gerçek linkler ve preview alanları

## Kurulum

Zip içindeki `portfolio-os-phase-6-real-content` klasörünün içeriğini mevcut projenin üstüne kopyala.

```bash
npm install
npm run dev
```

## Not

Bazı external siteler iframe içinde açılmayabilir. Bu normaldir; site kendi güvenlik ayarlarıyla iframe'i engelleyebilir. Link butonları yine çalışır.
